# Resources

Hackathons are great chances to build new skills. Here's a short list of suggested resources for the AI Systems Hackathon!

## 🤖 Query Generative AI

- ⚙️ [OpenAI API documentation](https://platform.openai.com/docs)
- 🐍 [Ollama](https://ollama.com/) — run LLMs locally; [Ollama Cloud](https://ollama.com/) for hosted options
- 📃 [Querying AI in R](https://posit.co/blog/) (e.g., `openai` or `ollama`-style packages)
- 📃 [Querying AI in Python](https://platform.openai.com/docs/libraries) (e.g., `openai`, `ollama` Python clients)
- 🔍 **Course materials**: SYSEN 5381 DSAI — Query AI / LAB_ai_reporter style exercises

## 🗄️ Using Supabase for PostgreSQL

- ⚙️ [Supabase](https://supabase.com/) — free tier for PostgreSQL
- 📃 [Supabase + R](https://supabase.com/docs/reference) (e.g., REST API or `DBI`/`RPostgres`)
- 📃 [Supabase + Python](https://supabase.com/docs/reference/python/introduction)
- 🔍 Quick start: Create a project, get connection string or API keys, run a simple query in R or Python in ~10 minutes

## 🌐 Building REST APIs

- 📦 [Build a REST API in R with Plumber](https://posit.co/blog/creating-apis-for-data-science-with-plumber/)
- 📃 [Plumber R package](https://www.rplumber.io/)
- 🔍 [**Template REST API**](https://github.com/timothyfraser/aisystemshackathon/tree/main/demos/api) (R/Plumber)
- 📃 [FastAPI](https://fastapi.tiangolo.com/) or [Flask](https://flask.palletsprojects.com/) for Python APIs

## 💻 Using Cursor IDE

- ⚙️ [Cursor](https://cursor.com/) — AI-supported editor
- 📃 Use Cursor for writing and refactoring code, generating docstrings, and debugging
- 🔍 **Course materials**: [SYSEN 5381 DSAI — productivity / Cursor rules and labs](https://github.com/timothyfraser/dsai/tree/main/02_productivity)

## 🚀 Deploying to Posit Connect

- 📦 [Posit Connect](https://posit.co/products/enterprise/posit-connect/) — deploy Shiny apps, APIs, and reports
- 📃 [Deploying to Posit Connect](https://docs.posit.co/connect/) — publish apps and APIs
- 🔍 **Hackathon**: Use the event's Posit Connect server (details at the Deploy training)

## 🐙 Using GitHub

- ⬇️ [How to Clone a Git Repository](https://youtu.be/JeDWlcFsFAA?si=ilCx1ApTAbQMO7vt)
- 🔑 [How to Get a Personal Access Token from GitHub](https://github.com/timothyfraser/aisystemshackathon/blob/main/docs/github_pat.md)

## 📚 Making Good Documentation and READMEs

- 📃 [Making Good READMEs](https://www.makeareadme.com/) — structure and best practices
- 📈 [Mermaid for diagrams in READMEs](https://mermaid.live)
- 💩 [Icons for your GitHub READMEs](https://github.com/timothyfraser/aisystemshackathon/blob/main/docs/icons.md)
