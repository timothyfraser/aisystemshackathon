## What is **DigitalOcean**?

> **DigitalOcean** is a cloud platform for deploying web apps, APIs, and databases. In this hackathon repo, we use it as a practical deployment option for multiple app types.

## What to Expect

- The smallest App Platform tier is usually low-cost and good for demos/prototypes.
- You can pause/archive infrastructure when not in use.
- You will need a valid payment method for live deployments.

## Deployment Targets in This Repo

Use DigitalOcean docs together with these demo folders:

- [`../fastapi/`](../fastapi/) - Python FastAPI API
- [`../plumber/`](../plumber/) - R Plumber API
- [`../shinypy/`](../shinypy/) - Shiny for Python app
- [`../shinyr/`](../shinyr/) - Shiny for R app
- [`../supabase/`](../supabase/) - Optional database backend workflow
